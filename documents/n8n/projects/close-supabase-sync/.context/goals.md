# Project Goals

## Primary Goal
Sync data from Close.io CRM into Airtable for sales analytics.

## Secondary Goals
- Analyze call recordings with OpenAI Whisper for quality review
- Build dashboards for sales performance, conversation quality, and process tracking
- Sync Demodesk meeting transcripts with talk ratio analysis
- Maintain user/lead/opportunity data in Airtable as single source of truth for reporting

## Key Metrics
- All Close.io custom activities synced daily to Airtable
- All opportunities with value, status, confidence tracked
- Call recordings transcribed and linked to leads/users
- Demodesk transcripts with talk ratios available for analysis
