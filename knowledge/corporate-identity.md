# Corporate Identity — Localweb

> Design-Richtlinien fuer alle generierten Dokumente (PPTX, DOCX, etc.)
> Zuletzt aktualisiert: 2026-03-25

---

## Farben

| Rolle | Hex | Verwendung |
|---|---|---|
| Primaer (Blau) | `#0693e3` | Ueberschriften, Buttons, Akzente |
| Sekundaer (Dunkelgrau) | `#32373c` | Text, Hintergruende, Footer |
| Akzent (Gruen) | `#00d084` | Highlights, Erfolgs-Elemente, CTAs |
| Weiss | `#FFFFFF` | Hintergrund, Kontrast |
| Schwarz | `#000000` | Fliesstext |

---

## Schrift

| Kontext | Font | Fallback |
|---|---|---|
| Ueberschriften | Arial | Helvetica, sans-serif |
| Fliesstext | Arial | Helvetica, sans-serif |
| Monospace (Code) | Consolas | Courier New, monospace |

## Logo

| Format | Pfad / URL |
|---|---|
| Web (PNG) | `https://localweb.de/wp-content/uploads/2025/06/cropped-Group-3.png` |

## Tonalitaet

- **Anrede:** Per Du
- **Stil:** Conversational, locker, direkt
- **Sprache Dokumente:** Deutsch
- **Sprache Code/Variablen:** Englisch

---

## PPTX-Vorlage

| Element | Stil |
|---|---|
| Titelfolie | Dunkelgrau-Hintergrund (`#32373c`), weisser Titel, Logo oben links |
| Inhaltsfolien | Weisser Hintergrund, Blau (`#0693e3`) fuer Ueberschriften |
| Akzent-Elemente | Gruen (`#00d084`) fuer Hervorhebungen, Icons |
| Footer | Dunkelgrau-Balken mit "localweb.de" in Weiss |
| Schriftgroessen | Titel: 36pt, Untertitel: 20pt, Body: 16pt, Footer: 10pt |

---

## DOCX-Vorlage

| Element | Stil |
|---|---|
| Deckblatt | Logo oben links, Titel in Blau (`#0693e3`), Linie in Gruen |
| Ueberschrift 1 | Blau (`#0693e3`), 18pt, Bold |
| Ueberschrift 2 | Dunkelgrau (`#32373c`), 14pt, Bold |
| Fliesstext | Schwarz, 11pt, Zeilenabstand 1.15 |
| Kopfzeile | Logo links, Dokumenttitel rechts (10pt, Grau) |
| Fusszeile | "localweb.de" links, Seitenzahl rechts |
| Akzente | Gruene Linie (`#00d084`) unter Ueberschriften |
